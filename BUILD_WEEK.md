# OpenAI Build Week development record

## Pre-existing foundation

Before July 13, 2026, AI Dock began as a personal always-on-top popup that embedded several AI websites and offered a local Qwen tab.

## Meaningful extensions during the submission period

- Universal MCP capability discovery and compact schema retrieval.
- Cloud-only semantic planning with adaptive provider reliability and circuit breakers.
- Multi-provider council planning and independent criticism for complex or risky plans.
- Explicit intent contracts preventing dropped clauses in compound requests.
- Recursive action-schema and authority validation.
- Crash-safe task journals, action checkpoints, `/resume`, idempotency and bounded recovery.
- Deterministic verification of files, folders, workspaces, windows and process outcomes.
- Persistent typed conversational references and correction learning.
- Normal logged-in Brave reuse and workspace-aware window merging.
- System-aware package identity resolution for CachyOS/Arch.
- Research, knowledge, development, document, media, data, operations and monitoring MCP families.
- A 54-check non-destructive regression suite and portable, sanitized distribution.

## Collaboration with Codex

Codex was used as the primary engineering collaborator to inspect the running system, translate product feedback into bounded capabilities, implement and debug the runtime, create regression tests from real failures, verify behavior on Hyprland, and prepare this public edition. The human supplied the product direction, real-world failure reports, workflow priorities, safety decisions and final acceptance criteria. Codex accelerated cross-file implementation, test generation, log analysis and repeated system-level validation.
